# Games101-Class-Notes
Games101课堂笔记（渲染、几何、光线追踪、动画模拟）--Astro WANG



本课程将全面而系统地介绍现代计算机图形学的四大组成部分：

（1）光栅化成像，（2）几何表示，（3）光的传播理论，以及（4）动画与模拟。
